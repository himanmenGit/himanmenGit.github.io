* 파이썬은 클래스 정의 이외에서 볼 수 없도록 하는 속성에 대한 네이밍 컨벤션이 있다. 
* 속성 이름앞에 두개의 언더스코어(__)[던더]를 붙이면 된다
```python
class Duck():
def __init__(self, i_name):
self.__name = i_name
@property
def name(self):
print('inside the getter')
return self.__name
@name.setter
def name(self, i_name):
print('inside the setter')
self.__name = i_name
```
```python
fow1 = Duck('Howard')
fow1.name
```
inside the getter
'Howard'
```python
fow1.name = 'Donald'
```
inside the setter
```python
fow1.name
```
inside the getter
'Donald'
```python
fow1.__name
```
